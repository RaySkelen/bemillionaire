# Who Wants to be a Millionaire

A simple "Who wants to be a millionaire" web-app made with React. 

You can check it at GitHub Pages: https://rayskelen.github.io/deploy/,
or run it on your local machine.

To run it: 
1. Clone this repository to your local machine. 
2. Run `npm i` or `yarn install` in your terminal. 
3. Run `npm start` or `yarn start` in your terminal.
